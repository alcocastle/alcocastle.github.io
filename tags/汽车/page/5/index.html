开完哈弗H9，再试驾三菱进口帕杰罗，对比后才知道差距明显

开完哈弗H9，再试驾三菱进口帕杰罗，对比后才知道差距明显。 长城哈弗H9和三菱进口帕杰罗，在越野e族中，都是属于可拿出手的越野车型，可以说哈弗H9就是国产硬派越野SUV的代表。 在售价上，哈弗H9 20.98万到27.28万的售价区间（售价为36.88万的2.0T罗伦式极星版车型暂时不说），与同是越野的牧马人、普拉多要便宜得多，满足了很多越野爱好者的需求。 而进口的三菱帕杰罗，售价34.98万起，与哈弗H9相比确实贵，但是帕杰罗在硬派越野SUV中也同样是售价比较亲民的，仅次于哈弗H9，那么30W左右的预算，买哈弗H9还是帕杰罗？ 总的来说，论哈弗H9和三菱帕杰罗的越野能力，懂车的人自然知道纯机械超选四驱的帕杰罗更好，不少车友这样评价“经常越野就选择帕杰罗，经常旅行跑高速偶尔越野以下选择H9比较好”。 不过与帕杰罗相比，哈弗H9的性价比是比较高的，既兼顾了家用还可以越野，但是论整车的可靠性、耐用性，H9与帕杰罗还有很大的差距，车友给出“先买山猫玩几年，再换全新的H9”的建议，也是不错的选择，毕竟目前哈弗H9的优惠力度也是比较大的。

揭秘特斯拉Model 3高性能版本的秘密

特斯拉 Model 3 的最强大版本提供了一种如此强烈以至于让人上瘾的体验。 特斯拉的情况很特殊。 这家美国公司虽然成为全球汽车行业日益强大的参与者，但不能以与其他制造商相同的条件来评判。 用特斯拉人自己的话来说，他们的汽车不是包含复杂电子系统的车辆，而是具有车身、驱动系统和车轮的复杂计算机。 较小的特斯拉车型的速度和道路行为给我们留下了深刻的印象。 我们还知道有一个更快的Performance版本，我们想知道它还能提供什么。 训练有素的眼睛的差异 从外部来看，Model 3 Performance与 Long Range 之间几乎没有什么区别。 否则，它与路上行驶的特斯拉 Model 3 的其余部分相同。 您会立即认识到特斯拉对其较小型号的“抽象”哲学。 Performance 与 Model 3 其余部分的不同之处在于操作系统中添加了选项以调整功率分配。 深深地踩下加速器，加速是灾难性的，并且没有丝毫延迟。 没有内燃机可以做到这一点，将车轮直接连接到驾驶员的右脚。 更运动的行为 Model 3 Performance 的第二个最基本的区别是不可见的。 的确，如果要做“流氓”并不是最合适的。 即使“我用电动”计划有 6,000 的折扣，这也不是一个小数目，但如果你想在这个价格类别中找到具有这种性能的汽车，人们根本找不到任何汽车。 然而，除了金钱之外，驾驶特斯拉的感觉仍然是一种非常特别、独特的体验。

不务正业的Mustang野马，不仅出了SUV，还是电动的

30万元左右的预算，打算购入一辆纯电动 SUV，很多人的第一想法就是特斯拉的Model Y，如果你不想随波逐流，想追求点个性。 不妨考虑一下福特的Mustang Mach-E。 有人说这不是福特野马吗？ 没错，不过野马也开始不务正业的，不仅出了 SUV，而且还是一辆电动 SUV，之所以采用Mustang的名字，因为当你第一眼看到它的时候，你就会被它的运动风范所吸引，就像看到那辆Mustang跑车一样。 虽然定位为 SUV 车型，但在Mustang Mach-E身上，我们可以发现很多历代Mustang跑车的设计元素，即便是一款纯电动车，封闭式的前脸依然“模拟”出了Mustang风格的格栅，高亮黑色的饰板和蜂窝状纹理，乍一看有一种燃油车的感觉。 内饰部分，利刃式的尾灯和大部分纯电车型一样采用了简约的设计理念，全系标配的12.2英寸液晶仪表和15.5英寸的竖状中控大屏，以及三幅平底式多功能方向盘几乎集成了车内的大部分操作和显示功能。 车身尺寸上，Mustang Mach-E的长、宽、高分别为4730、1886、1621（1613）mm，轴距为2984mm，这一点和 Model Y 是不相上下的，同时接近3 m的轴距为其带来了充裕的车内乘坐空间，以身高175cm的成年男性体验者为例，坐进后排，头部空间大约剩余一拳，腿部空间大约剩余两拳。
