酸菜豆腐粉条炖骨头，东北人都爱吃，连汤带肉全吃光 几乎没有一个东北人不喜欢吃酸菜，尤其是选择在外打拼的东北人，想家的时候最思念的就是酸菜的味道。 酸菜就是极具代表的东北味儿，最早北方人选择制作酸菜，也是为了将食物可以将食物保存起来以供冬日食用。 “请君伐菹薪，煮沸水为盐”，《管子》一书中将制作酸菜的方法说的很清楚，而菹这个字最早说的就是酸菜。 其实家常菜吃的就是随意，食材上也大多不大讲究，像这道酸菜豆腐粉条炖腔骨，就有着东北大乱炖的基因在里面。 酸菜和腔骨经过炖煮，酸菜吸收了腔骨的香味，又因为本身酸菜就有解腻的酸味，让腔骨又多了一些滋味在里面。 炖煮的时候放上一点豆腐和粉条，虽然是配菜但是味道也是极好的，再喝点炖出来的骨头汤真的是美翻了。 【菜名】:酸菜豆腐粉条炖腔骨 【所需配料】:腔骨、酸菜、豆腐、粉条 【烹饪过程】： 步骤一、不同部位的猪骨有不同的味道，炖酸菜用腔骨或者是大骨棒都非常好，新买的腔骨清水冲洗干净，放在一个大锅里，注入大量的清水开火将腔骨中的血沫煮出来。 步骤六、将处理好的酸菜和豆腐倒入砂锅中，继续炖煮。 或者盛出一些锅中的汤来泡粉条也可以，泡软之后的粉条下到锅里，很快就会煮好。 步骤九、基本上粉条煮到晶莹剔透就做好了，这一锅酸菜豆腐粉条炖腔骨相当美味。 4、炖出来的骨头汤也很美味，加上一点胡椒粉喝完之后身上很暖和，天冷的时候吃这道酸菜豆腐粉条炖腔骨非常合适。
