揭秘特斯拉Model 3高性能版本的秘密

2021-11-05 · 1 min

特斯拉 Model 3 的最强大版本提供了一种如此强烈以至于让人上瘾的体验。 特斯拉的情况很特殊。 这家美国公司虽然成为全球汽车行业日益强大的参与者，但不能以与其他制造商相同的条件来评判。 用特斯拉人自己的话来说，他们的汽车不是包含复杂电子系统的车辆，而是具有车身、驱动系统和车轮的复杂计算机。

较小的特斯拉车型的速度和道路行为给我们留下了深刻的印象。 我们还知道有一个更快的Performance版本，我们想知道它还能提供什么。 训练有素的眼睛的差异 从外部来看，Model 3 Performance与 Long Range 之间几乎没有什么区别。 否则，它与路上行驶的特斯拉 Model 3 的其余部分相同。

您会立即认识到特斯拉对其较小型号的“抽象”哲学。 Performance 与 Model 3 其余部分的不同之处在于操作系统中添加了选项以调整功率分配。 深深地踩下加速器，加速是灾难性的，并且没有丝毫延迟。 没有内燃机可以做到这一点，将车轮直接连接到驾驶员的右脚。 更运动的行为 Model 3 Performance 的第二个最基本的区别是不可见的。 的确，如果要做“流氓”并不是最合适的。 即使“我用电动”计划有 6,000 的折扣，这也不是一个小数目，但如果你想在这个价格类别中找到具有这种性能的汽车，人们根本找不到任何汽车。 然而，除了金钱之外，驾驶特斯拉的感觉仍然是一种非常特别、独特的体验。
